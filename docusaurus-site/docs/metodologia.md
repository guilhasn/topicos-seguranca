# Metodologia por ficha

Cada ficha deve seguir este formato:

1. Objetivos de aprendizagem
2. Codigo final de referencia
3. Classes, metodos, propriedades e funcoes usadas
4. Explicacao linha a linha
5. Pontos de seguranca
6. Erros comuns
7. Mini desafio

## Template rapido

~~~markdown
## Objetivos
- ...

## Codigo de referencia
```csharp
// codigo aqui
```

## Mapa de conceitos C# desta ficha

### Classe: NomeDaClasse
- O que e: ...
- Para que serve: ...

### Metodo: NomeDoMetodo(...)
- O que faz: ...
- Parametros: ...
- Valor devolvido: ...

### Propriedade: objeto.NomePropriedade
- O que guarda: ...
- Exemplo de uso: ...

### Funcao (metodo com retorno): NomeDaFuncao(...)
- O que devolve: ...
- Onde e usada: ...

## Explicacao linha a linha
1. ...

## Pontos de seguranca
- ...

## Erros comuns
- ...

## Mini desafio
- ...
~~~

Este formato permite manter consistencia entre as 9 fichas e facilita o estudo autonomo.
Regra didatica: explicar com linguagem simples, curta e direta, como se fosse uma aula guiada.
