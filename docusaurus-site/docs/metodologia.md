# Metodologia por ficha

Cada ficha deve seguir este formato:

1. Objetivos de aprendizagem
2. Codigo final de referencia
3. Explicacao linha a linha
4. Pontos de seguranca
5. Erros comuns
6. Mini desafio

## Template rapido

```markdown
## Objetivos
- ...

## Codigo de referencia
```csharp
// codigo aqui
```

## Explicacao linha a linha
1. ...

## Pontos de seguranca
- ...

## Erros comuns
- ...

## Mini desafio
- ...
```

Este formato permite manter consistencia entre as 9 fichas e facilitar o estudo autonomo.
